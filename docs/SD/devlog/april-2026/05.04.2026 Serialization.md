Implemented *Serialization* and *Deserialization* for components, ECS and commands/command queues.

This is useful for both saving information to files and to use with networking. 

The [serializer](https://sd.vatnar.dev/docs/classSD_1_1Serializer.html) class writes and reads from a `std::vector` of `std::bytes`'s. A member function [GetSpan()](https://sd.vatnar.dev/docs/classSD_1_1Serializer.html#af48a4d9c8f99ac0e8054918cf0b19671) . That can be used with utility functions [ReadBinary()](https://sd.vatnar.dev/docs/namespaceSD_1_1Filesystem.html#a32076096f28fef069c08399db6ac04b2) and [WriteBinary()](https://sd.vatnar.dev/docs/namespaceSD_1_1Filesystem.html#a754822ca3712f9326527d79a231372b6).

To ensure that instances of the ECS can be serialized, added components must be a [SerializableComponent (concept)](https://sd.vatnar.dev/docs/conceptSD_1_1SerializableComponent.html).
