Automated deployment of doxygen generated docs at https://sd.vatnar.dev/docs/ and this development journal containing devlogs and design decisions, and future guides for usage. It deploys automatically on pushes to github using github actions, building documentation from inline documentation with doxygen, and using quark to build the obisidan vault for more written documentation like this.

Due to this being done quite awhile after the start of the project, devlogs for the start of the project are missing, but I will try to update it each time I work.
